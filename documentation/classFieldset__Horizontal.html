Generates multiple rows as a group.

Displays multiple copies of a single row in a list-style format. All rows must be part of the same database table. Titles for each field are shown above all the rows, with each row following on its own line. This form makes use of the form's optional parameter by adding the specified number of extra empty rows to the end of the list. If this field is left blank or 0, no extra rows are added.

Parameters:
OBJECT $iterator The row manager to use as a template for all rows.
ARRAY $fields Array of field objects (database fields) to include in the form.
OBJECT $form Form object these fieldsets will belong to.
