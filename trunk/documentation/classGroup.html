Detailed Description

Handles display and processing for a Group field.

A group field is a set of fields that are combined together using a standard delimiter. Date fields are a good example of a group field (ex. month day year with a dilimter of '/') Page ranges are another example (start end with a delimiter of '-') $options["elements"] - number of individual fields to display $options["seperator"] - dilimiter for the fields $options["size"] - size of the form's text input fields OPTIONAL $options["maxlength"] - maximum length of text in any given input field $options["prefix"] - A prefix to prepend to the field data after it has been combined $options["suffix"] - A suffix to append to the field data after it has been combined
